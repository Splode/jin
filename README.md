## Usage

### Write Notes

```bash
$ jin add [notebook] <note>
# jin a [notebook] <note>
```

```bash
$ jin add [notebook]
# jin a [notebook]
```

#### Example

```bash
$ jin add laravel "Use 'php artsisan migrate:refresh' to rollback and recreate migrations"
```

### List Notes

```bash
$ jin list [notebook]
# jin ls [notebook]
```

```bash
$ jin list
# jin ls
```
